# 📸 截图说明

本目录用于存放项目截图。

## 添加截图

1. 将截图保存到本目录，命名格式：
   ```
   screenshot-功能名称-序号.png
   ```

## 推荐截图

- [ ] 首页截图 - 展示整体界面
- [ ] API 文档截图 - Swagger UI 展示
- [ ] 对话示例截图 - 展示实际对话效果
- [ ] 知识库上传截图 - 展示文档上传功能
- [ ] Docker 部署截图 - 展示容器运行状态

## 截图尺寸建议

- 宽度: 1200px - 1600px
- 格式: PNG 或 JPG
- 质量: 高清，无压缩

## 示例

```
docs/
├── USER_GUIDE.md
├── screenshots/
│   ├── screenshot-homepage-01.png
│   ├── screenshot-api-docs-01.png
│   └── screenshot-chat-demo-01.png
└── ...
```
